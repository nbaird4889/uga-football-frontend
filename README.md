# uga-football-players

## UGA FOOTBALL
After winning a national champtionship this year, this site highlights some of the UGA football legends.  

## Technologies Used 
- HTML
- Express
- React
- CSS
- Javascript 
- NodeJS
- MongoDB/Mongoose
